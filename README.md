Instructivo de navegación del aplicativo web para la gestión financiera de la empresa KASALION S.A.S.

1. Acceso al sistema
Para ingresar al aplicativo web, el usuario debe acceder a la URL sgestionfinanciera-kasalion.up.railway.app proporcionada por Railway(hosting) e ingresar sus credenciales.
Una vez autenticado correctamente, el sistema redirige automáticamente al panel principal (dashboard), mostrando en el menú lateral izquierdo las opciones disponibles según 
el rol asignado (Administrador o Auxiliar).

2. Estructura general del sistema
El aplicativo está organizado en un menú de navegación principal que contiene los siguientes módulos:

Gestión de usuarios (solo administrador).
Reportes (solo administrador).
Ventas.
Productos.
Ingresos.
Egresos.

Cada módulo permite realizar acciones específicas relacionadas con la gestión financiera de la empresa.

3. Navegación por módulos
   3.1. Gestión de productos
   Permite registrar, visualizar y eliminar productos.
   El usuario debe ingresar la información requerida (nombre, categoría, precio, etc.) y guardar el producto.

   3.2. Registro de ventas
  En este módulo se registran las ventas realizadas.
  El usuario debe ingresar los datos del cliente, seleccionar los productos que el cliente adquirió y confirmar la venta, el sistema calcula automáticamente el valor total.

  3.3. Registro de ingresos
  Permite registrar los pagos asociados a ventas previamente creadas, el usuario selecciona la venta correspondiente y el sistema asocial el valor recibido, 
  luego hace clic en guardar.
  En este campo el usuario podrá ver los ingresos registrados.

  3.4. Registro de egresos
  Permite registrar los egresos, el usuario selecciona la categoría del egreso y el valor del egreso, luego hace clic en guardar.
  En este campo el usuario podrá ver los ingresos registrados.

  3.5. Reportes (solo para administrador)
  Permite generar reportes financieros seleccionando un periodo específico (mensual o quincenal 1-15).
  El sistema consolida los ingresos y egresos y permite descargar el archivo correspondiente en formato .xlsx.

  3.6. Gestión de usuarios (solo para administrador)
  
      3.6.1. Crear nuevos usuarios
      El sistema permite crear nuevos usuarios y asignar roles/permisos.
      
      3.6.2. Administrar usuarios
      El sistema permite modificar los permisos o eliminar usuarios existentes

El aplicativo ha sido diseñado con una interfaz intuitiva y de fácil navegación, permitiendo a los usuarios interactuar con los diferentes módulos sin necesidad de conocimientos técnicos avanzados.
Este instructivo proporciona una guía básica para comprender el funcionamiento general del sistema.
